## reducer(state, action)
root-reducer

combines the following reducers into a single one:
* api
* comments
* desktopApp
* entries
* eventInfo
* events
* eventScreenshots
* images
* user

**Kind**: global function  

| Param | Type |
| --- | --- |
| state | <code>object</code> | 
| action | <code>object</code> |
